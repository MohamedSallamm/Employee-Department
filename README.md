This is a simple ASP.NET MVC project designed to manage company departments and employees. 
The application allows users to perform CRUD (Create, Read, Update, Delete) operations on departments and employees.

Key functionalities include:
- Adding, editing, and deleting departments.
- Adding, editing, and deleting employees and assigning them to departments.
- Basic data validation and error handling.

## Technologies Used
- ASP.NET MVC 
- C#
- Entity Framework
- SQL Server (LocalDB)
- Bootstrap  (for basic UI styling)

## Features
- Manage departments (CRUD operations)
- Manage employees and link them to departments
- User-friendly interface with Bootstrap
- Validation for required fields
